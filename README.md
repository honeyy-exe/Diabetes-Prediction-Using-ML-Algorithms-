# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview
This project aims to predict whether a person is diabetic or not using machine learning algorithms. It involves data preprocessing, exploratory data analysis (EDA), and comparison of multiple models to identify the best-performing classifier. :contentReference[oaicite:0]{index=0}

---

## 🎯 Objective
To build and evaluate machine learning models that can accurately predict diabetes based on medical and lifestyle features.

---

## 📊 Dataset
The dataset contains health-related features such as:
- Glucose Level
- Blood Pressure
- BMI (Body Mass Index)
- Insulin
- Age
- Skin Thickness
- Diabetes Pedigree Function

👉 Target Variable:
- `0` → Non-diabetic  
- `1` → Diabetic  

---

## ⚙️ Project Workflow

### 🔹 Data Preprocessing
- Handling missing values
- Data cleaning and normalization
- Feature selection

### 🔹 Exploratory Data Analysis (EDA)
- Correlation heatmaps
- Feature distributions
- Data visualization

### 🔹 Model Training
Implemented multiple machine learning algorithms:
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes :contentReference[oaicite:1]{index=1}

---

## 📈 Model Evaluation
Models were evaluated using:
- Accuracy Score
- Confusion Matrix
- Precision, Recall, F1-score

👉 Best model selected based on performance comparison.

---

## 🧠 Key Insights
- Glucose and BMI are strong indicators of diabetes
- Some features show correlation and redundancy
- Ensemble models generally perform better
